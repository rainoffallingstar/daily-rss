The potential of federated learning for self-configuring medical object detection in heterogeneous data distributions

Medical Object Detection (MOD) is a clinically relevant image processing method that locates structures of interest in radiological image data at object-level using bounding boxes. High-performing MOD models necessitate large datasets accurately reflecting the feature distribution of the corresponding problem domain.
