Virtual Orientation Tools (VOTj): Fiji plugins for object centering and alignment

Standardizing image datasets is essential for facilitating overall visual comparisons and enhancing compatibility with image-processing workflows. One way to achieve homogeneity for images containing a single object is to align the object to a common orientation. Here, we propose the Virtual Orientation Tools (VOTj): a set of Fiji plugins to center and align an object of interest in images to a vertical or horizontal orientation.
